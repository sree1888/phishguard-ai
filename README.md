# PhishGuard AI

AI-based phishing detection web application.

## Features
- Detects phishing messages
- Risk score analysis
- Cybersecurity-themed UI
- Real-time threat detection

## Tech Stack
- Python
- Flask
- HTML/CSS

## How It Works
Users paste suspicious emails, SMS, or URLs.
The system analyzes suspicious keywords and calculates a phishing risk score.

## Future Scope
- AI integration
- Browser extension
- Real-time URL scanning
